Hello Blog

 2018/08/22

努力克服大脑中袭来的懒惰，打开音乐播放器，调整好状态来写这篇拖延了一年之久的文章，没有错，自从博客搭建完成后，这一年来我没有产出一篇文章，仔细想想，恐怕自己的自控力已经跌破了基准值。完成这些文章，就算是对拖延症大魔王发起的第一次挑战吧。

就好像程序员接触新语言的时候第一步要输出一个Hello World一样，开始一件新的事情的时候，做一个小小的仪式，内心也会严肃起来吧。仍然记得一年之前在查找关于搭建博客的资料的时候无意中看到一篇叫做《我为什么要写博客》的文章，详细的内容已经记不大清楚了，只留下了一丝丝感动和敬重。之后我也多次关注过一些人的博客，有的是翻译文章，还有些是关于学习的总结，也是因此坚定了要写一些东西和其他人分享的想法。

在搭建独立的个人博客之前，大学期间也曾在Lofter上发过当时学习photoshop临摹的一些小图标，也经常使用豆瓣记录自己读过的书和看过的电影，不过从未想过自己在网络上输出的这点信息会有多少人留意，毕竟网络这么大。在一次外出旅行之后，因为拍摄了不少照片又无人可分享，就在豆瓣上用照片堆砌了一篇图文，没想到不久之后会被越来越多的人浏览、留言、收藏。大概就是被人认可了吧，觉得特别开心，意识到自己可以把喜欢的东西分享给更多的人，分享自己的想法。

渐渐萌生了想要在网络中拥有自己的一丝空间的想法，独立于其他平台之外的小空间，最后决定在Github上搭建一个静态博客。尽管还没有考虑好下一步要怎么做，不过只要在路上就好。

就是这样，聆空的博客诞生了，发布这篇文章的时间就是skyorbit.xyz的生日🎉🎉🎉

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: false
    tags: true
